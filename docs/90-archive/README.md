# Archive

这里存放 **历史设计、旧规划、单次发布记录**。

这些文档的用途只有三个：

- 查当时为什么这么设计
- 查某个方案是如何演进的
- 查一次性发布记录

默认不要把这里当作当前实现真源。

## 1. 归档区结构

- [historical-design/README.md](./historical-design/README.md)
  - 早期设计稿、早期项目结构说明
- [feature-plans/README.md](./feature-plans/README.md)
  - 已落地或已废弃的功能规划文档
- [release-notes/README.md](./release-notes/README.md)
  - 单次版本说明

## 3. 使用规则

- 先读主线文档，再回头查归档
- 归档文档里的旧结论不能直接覆盖代码真相
- 如果发现归档文档里有仍然有效的稳定知识，应提炼回主线 docs
