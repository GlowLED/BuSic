# Release Notes Archive

这里存放单次版本说明。

它们可以回答：

- 某个版本当时发布了什么
- 某个阶段有哪些显式功能变化

它们不能回答：

- 当前发布流程怎么走
- 当前更新系统如何工作
- 当前 Release 资产命名规则是什么

这些问题请优先看：

- [../../20-workflows/release-workflow.md](../../20-workflows/release-workflow.md)
- [../../10-project/update-system.md](../../10-project/update-system.md)
